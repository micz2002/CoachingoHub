# CoachingoHub

## Overview
CoachingoHub is a **CRM system for personal trainers** in the form of a web application. The application is dedicated to personal trainers, especially those managing a large group of clients. The system supports comprehensive client management, including training planning, appointment scheduling, and notification handling.

It enables the automation of administrative processes, work organization, and improved communication.

## Features
✅ Comprehensive Client Management – Easily add, remove, and modify client profiles, track their progress, and store training notes.

✅ Smart Scheduling – Set up and manage training appointments, view schedules, and allow clients to cancel or reschedule sessions.

✅ Dashboard Analytics – Get insightful statistics about training sessions, clients, and scheduling trends.

✅ Automated Notifications – SMS and email reminders for upcoming training sessions to keep clients engaged.

✅ Secure Authentication – JWT-based authentication system ensuring secure user access with role-based permissions.

✅ Training Reports – Clients can submit training reports, which trainers can review and track progress.

## Installation
### Backend (Spring Boot)
```bash
git clone https://github.com/yourusername/CoachingoHub.git
cd CoachingoHub
mvn clean install
mvn spring-boot:run
```

### Frontend (React)
```bash
cd frontend
npm install
npm start
```

## Tech Stack
- Frontend: **HTML, CSS, Java Script, React.js**
- Backend: **Spring Boot, Java, REST API**
- Database: **MySQL**
- Security: **JWT authentication**
- Notifications: **SMS via Vonage, email notifications**
